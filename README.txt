COMO RODAR O SISTEMA
--------------------

1. Instale o Python (versao 3.8 ou superior)
   Download em: https://www.python.org/downloads/

2. Abra o terminal/prompt na pasta do projeto

3. Instale as dependencias:
   pip install -r requirements.txt

4. Execute o sistema:
   
   Opcao 1 (Windows): 
   De duplo clique no arquivo INICIAR_SERVIDOR.bat
   
   Opcao 2 (Terminal):
   python INICIAR.py

5. O navegador abre automaticamente em:
   http://127.0.0.1:5000
